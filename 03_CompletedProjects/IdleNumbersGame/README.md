# Incremental Clicker Game

A test incremental game built using **Pygame**. Players can earn money by clicking buttons, purchase upgrades to increase earnings, and hire managers to automate tasks. 
---

## How to Run

1. **Clone the project and install dependencies**:
   Ensure you have Python and Pygame installed. You can install Pygame using pip:
   ```bash
   pip install pygame
   ```

2. **Run the Game**:
   Open a terminal in the project directory and run:
   ```bash
   python clickerTutorial_pygame.py
   ```

3. **Play**:
   - Click the buttons to earn money.
   - Use the earned money to buy upgrades and hire managers.
   - Watch your earnings grow!


## Controls

- **Mouse Click**: Interact with buttons to earn money, purchase upgrades, or hire managers.
- **Quit**: Close the game window to exit.
